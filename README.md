**SQL Practice Repository**

**Overview88

This repository is for practicing SQL querying using Python (e.g., sqlite3, pandas.read_sql). It includes scripts and notebooks for writing and testing SQL queries for data analysis.

**Structure**

**/notebooks/:** Jupyter Notebooks for SQL queries (e.g., customer_queries.ipynb).

**/scripts/:** Python scripts for SQL-related functions.

**/data/:** SQLite databases or datasets (not tracked; store locally).

**/images/:** Visualizations of query results.


**Setup Instructions**


    Clone the Repository:git clone https://github.com/anikashraful/sql-practice.git


    Install Dependencies:pip install -r requirements.txt


**Run Notebooks:**Open .ipynb files in /notebooks/ using Jupyter Notebook or VS Code.


**Requirements**

Python 3.8+

Libraries: sqlite3 (built-in), pandas, jupyter

Install via: tem:pandas

jupyter


.gitignore: Use the common .gitignore above.

**How to Add Files Later:**

Add a notebook with SQL queries (e.g., notebooks/customer_queries.ipynb).

    Commit:git add notebooks/customer_queries.ipynb

    git commit -m "Add SQL queries for customer data"

    git push origin main

**# Deep Learning Practice Repository**

**Overview**

This repository is for practicing deep learning using PyTorch in Python. It includes Jupyter Notebooks and scripts for neural networks, CNNs, RNNs, and NLP tasks.

**Structure**

**/notebooks/:** Jupyter Notebooks for DL models (e.g., mnist_cnn.ipynb).

**/scripts/:** Python scripts for reusable DL functions.

**/data/:** Datasets (not tracked; store locally or link to sources).

**/images/:** Visualizations of model performance.


**Setup Instructions**

    **Clone the Repository:**git clone https://github.com/anikashraful/deep-learning-practice.git


    Install Dependencies:pip install -r requirements.txt


    Run Notebooks:Open .ipynb files in /notebooks/ using Jupyter Notebook or VS Code.

**Requirements**

Python 3.8+

Libraries: torch, pandas, numpy, matplotlib, jupyter
